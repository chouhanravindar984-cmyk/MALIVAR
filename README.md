# MALIVAR

MALIVAR is an experimental project for running Windows apps
on Android devices powered by MediaTek and Mali GPUs.

Version: V1.0 (Foundation)

This project is for learning, testing, and community experiments.
## MALIVAR V1.0 Goals

- MediaTek + Mali devices only
- Basic Windows apps support
- No heavy PC games
- Focus on stability, not performance

MALIVAR is based on the Winlator project and focuses on
MediaTek + Mali GPU optimization.
MALIVAR V1.0 uses a custom Winlator container profile
designed for MediaTek devices with Mali GPUs.
